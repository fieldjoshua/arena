How to Compete

  1. Pick a challenge and read the spec
  2. Solve it with any AI tools you want — Cursor, Claude, Copilot, all of them
  3. Write your technique.json (category, tools, tokens, time)
  4. Submit via PR to submissions/<challenge-id>/<handle>/
  5. CI auto-scores and posts results on your PR
  6. Merge updates the leaderboard

Scoring

Composite = Correctness (40%) + Token Efficiency (20%) + Wall Time (20%) + Solution Size (10%) + API Cost (10%)

Technique signatures (tools used, token counts, session logs) are tracked and displayed on the leaderboard but do not affect scoring in v0. The community will vote on when and how technique data enters the composite formula.
